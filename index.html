Background Image

Background-Size mit background-size können wir die größe des bildes definieren, dies bezieht sich NICHT auf die größe des geladenen bildes, sondern darauf, wie groß unser bild angezeigt wird, wenn es kleiner wird als die box, wird es im standardfall wiederholt. der default wert ist background-size: auto; und gibt das bild in der größe aus, in der es angelegt wurde.


-background-size: cover; komplette Fläche wird horizontal ausgefüllt, auch wenn etwas abgeschnitten wird


-background-size: contain; komplette Fläche wird vertikal abgeschnitten, auch wenn etwas wiederholt wird


-background-repeat: repeat; Bild wird in alle Richtungen wiederholt!
background-repeat: repeat-x; Bild wird auf x-Achse wiederholt; analog dazu repeat-y, auf y-Achse wiederholt
background-repeat: no-repeat; keine Wiederholung
background-repeat: space; Bild breitet sich innerhalb seines Elternelements aus; der Platz wird so ausgefüllt, ohne abgeschnitten zu werden!
background-repeat: round Bild nutzt den Platz der Box so sehr, das es passt ohne Lücken zu hinterlassen.



Background-position: center (x-y-Achse zentriert); right center; left top; ...


Background-blend mode: soft light;... MDN Ressource


Shortcode background: farbe url wiederholung position größe;
background: tomate url repeat left-bottom 100px;
